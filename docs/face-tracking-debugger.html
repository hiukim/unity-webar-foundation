Face Tracking Debugger

To ease development, WebAR Foundation comes with a debugger for face tracking which can run in Editor.

1. Run the project in editor mode. You should see a randomized texture to simulate the webcam video feed.

2. To open the debugger panel, go to "Window" -> "WebAR" -> "Face Tracking Debugger"

3. trigger `Face Found` and `Face Lost` events.
